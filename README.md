# Gulp starter

### A workflow to start your project with sass and es6.

## Features:

### Styles
  * Compile sass
  * Auto indent

### Scripts
  * Compile ES6
  * Auto indent

## Who use:

```
cd Gulp-starter
```
```
npm install
```

when you code, run this:

```
gulp
```

if you are lost in your indentation, run this:

```
gulp autoIndent
```

When you're done, minify everything:

```
gulp prod
```

and your ready-to-go site is in the dist folder.
